## Overview

The HKEX data source fetches comprehensive company information from the Hong Kong Stock Exchange, including HSIC (Hang Seng Industry Classification) data, listing categories, and detailed quote information.

<Info>
  **Data Source**: HKEX official widget API at `www1.hkex.com.hk`
  
  **Coverage**: ~2,500 securities on HKEX Main Board and GEM
</Info>

## Architecture

The HKEX fetcher uses a two-stage approach:

1. **Webhook for Company List**: Fetches the full company list and authentication token
2. **Quote API for Enrichment**: Fetches detailed information per company using JSONP

### Webhook Endpoint

```python
# Webhook returns normalized stocklist and authentication token
WEBHOOK_URL = "https://n8n-auto.deltablock.io/webhook/996f70f0-..."
```

<Note>
  The webhook normalizes HKEX's complex nested response structure into a flat stocklist format.
</Note>

### Quote API

```python
QUOTE_URL = "https://www1.hkex.com.hk/hkexwidget/data/getequityquote"
```

**Provides:**
- HSIC industry and sub-sector classifications
- Listing category (Primary, GEM, Trading Only)
- Listing date, ISIN, lot size
- Company chairman, incorporation location

## Webhook Response Normalization

The webhook returns a nested structure that gets normalized:

```python
# From hkex_company_list.py:56-84
def _find_stocklist_container(self, payload: Any) -> Optional[dict]:
    """Find the stocklist container in nested payload."""
    stack = [payload]
    
    while stack:
        current = stack.pop()
        if isinstance(current, list):
            for item in current:
                if isinstance(item, (dict, list)):
                    stack.append(item)
            continue
        
        if not isinstance(current, dict):
            continue
        
        stocklist = current.get("stocklist")
        if isinstance(stocklist, list):
            return current
        
        for value in current.values():
            if isinstance(value, (dict, list)):
                stack.append(value)
    
    return None

def _normalize_hkex_payload(self, payload: Any) -> Optional[dict]:
    """Normalize webhook payload to standard format."""
    container = self._find_stocklist_container(payload)
    if not container:
        return None
    return {"data": container}
```

<Info>
  The normalization uses depth-first search to locate the stocklist regardless of webhook response structure changes.
</Info>

## Token Management

HKEX requires authentication tokens for quote API access:

```python
# From hkex_company_list.py:98-131
def _find_token_in_payload(self, payload: Any) -> Optional[str]:
    """
    Extract authentication token from webhook response.
    Token can be in multiple locations depending on response format.
    """
    # Check top-level token field
    if isinstance(payload, dict):
        direct = payload.get("token")
        if isinstance(direct, str) and direct.strip():
            return direct.strip()
        
        # Check requestUrl parameter
        request_url = payload.get("requestUrl")
        if isinstance(request_url, str) and request_url.strip():
            parsed = self._extract_token_from_request_url(request_url)
            if parsed:
                return parsed
    
    # Fallback: DFS through entire payload
    stack = [payload]
    while stack:
        current = stack.pop()
        if isinstance(current, dict):
            tok = current.get("token")
            if isinstance(tok, str) and tok.strip():
                return tok.strip()
            # Continue searching...
    
    return None
```

<Warning>
  Tokens expire periodically. The fetcher automatically refreshes tokens when quote API requests fail.
</Warning>

## JSONP Response Parsing

The HKEX Quote API returns JSONP format:

```python
# From hkex_company_list.py:163-187
def _parse_jsonp(self, response_text: str, callback_name: str) -> dict:
    """
    Parse JSONP response to extract JSON data.
    
    Args:
        response_text: The raw JSONP response
        callback_name: The callback function name used in the request
    
    Returns:
        Parsed JSON data as a dictionary
    """
    # Match pattern: callbackName({...})
    pattern = rf"{re.escape(callback_name)}\((.*)\)"
    match = re.search(pattern, response_text, re.DOTALL)
    
    if match:
        json_str = match.group(1)
        return json.loads(json_str)
    else:
        # Try a more generic pattern for any callback
        pattern = r"\w+\((.*)\)$"
        match = re.search(pattern, response_text, re.DOTALL)
        if match:
            return json.loads(match.group(1))
        raise ValueError("Could not parse JSONP response")
```

## Enrichment Process

```python
# From hkex_company_list.py:326-394
def enrich_with_quote_details(
    self, 
    companies: list[dict], 
    max_workers: int = 5,
    delay_per_request: float = 0.1,
    progress_callback=None
) -> list[dict]:
    """
    Enrich company list with detailed quote information.
    
    Args:
        companies: List of company dictionaries
        max_workers: Number of parallel workers for fetching
        delay_per_request: Delay in seconds between requests
        progress_callback: Optional callback function for progress updates
    
    Returns:
        Enriched list of companies with industry classifications
    """
    total = len(companies)
    completed = 0
    lock = threading.Lock()
    
    def fetch_and_enrich(company: dict) -> dict:
        nonlocal completed
        
        # Add delay to avoid rate limiting
        time.sleep(delay_per_request)
        
        symbol = company["symbol"]
        details = self.fetch_quote_details(symbol)
        
        if details:
            company.update({
                "industry": details.get("hsic_ind_classification"),
                "sector": details.get("hsic_sub_sector_classification"),
                "listing_category": details.get("listing_category"),
                "listing_date": details.get("listing_date"),
                "isin": details.get("isin"),
                "lot": details.get("lot"),
            })
        else:
            company.update({
                "industry": None,
                "sector": None,
                "listing_category": None,
                "listing_date": None,
                "isin": None,
                "lot": None,
            })
        
        with lock:
            completed += 1
            if progress_callback:
                progress_callback(completed, total)
        
        return company
    
    # Use ThreadPoolExecutor with rate limiting
    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        futures = {executor.submit(fetch_and_enrich, c): c for c in companies}
        enriched = []
        for future in as_completed(futures):
            enriched.append(future.result())
    
    # Sort back by symbol
    enriched.sort(key=lambda x: int(x["symbol"]) if x["symbol"].isdigit() else float('inf'))
    
    return enriched
```

<Note>
  HKEX enrichment requires 100ms delay per request (vs 50ms for SGX) due to stricter rate limiting.
</Note>

## Market Cap Calculation

HKEX returns market cap with unit suffixes:

```python
# From hkex_company_list.py:299-306
mktcap_raw = self._parse_float(stock.get("mktcap", ""))
mktcap_unit = stock.get("mktcap_u", "")
market_cap = None
if mktcap_raw is not None:
    unit_multipliers = {"K": 1_000, "M": 1_000_000, "B": 1_000_000_000}
    multiplier = unit_multipliers.get(mktcap_unit, 1)
    market_cap = mktcap_raw * multiplier
```

<Info>
  Market cap is automatically normalized to base values (e.g., 365.42B → 365420000000.0)
</Info>

## Error Handling

Automatic token refresh on failure:

```python
# From hkex_company_list.py:239-254
try:
    response = self.session.get(self.QUOTE_URL, params=params, timeout=15)
    response.raise_for_status()
    data = self._parse_jsonp(response.text, callback_name)
except (requests.RequestException, json.JSONDecodeError, ValueError):
    # One retry after refreshing token from webhook
    self.fetch_companies_from_webhook()
    if not self._quote_token:
        return None
    params["token"] = self._quote_token
    try:
        response = self.session.get(self.QUOTE_URL, params=params, timeout=15)
        response.raise_for_status()
        data = self._parse_jsonp(response.text, callback_name)
    except (requests.RequestException, json.JSONDecodeError, ValueError):
        return None
```
