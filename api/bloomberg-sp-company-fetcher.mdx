## Overview

The `BloombergSPCompanyFetcher` class provides methods to fetch company background data from Bloomberg's API using Singapore Exchange (SGX) symbols. It converts SGX symbols to Bloomberg's ticker format (e.g., `D05:SP`) and retrieves additional company information such as market capitalization, shares outstanding, and founding year.

## Class Definition

```python
class BloombergSPCompanyFetcher:
    """Fetches SGX company background data from Bloomberg."""
```

## Initialization

### `__init__()`

Initialize the fetcher with default headers for Bloomberg API.

```python
fetcher = BloombergSPCompanyFetcher()
```

The constructor automatically sets up:
- A requests session with appropriate headers for Bloomberg API access
- User agent headers for browser compatibility
- Accept headers for JSON responses
- Referer set to Bloomberg markets portal
- Thread-safe debug info storage for request troubleshooting

## API Endpoint

The class uses Bloomberg's quote API:

- `BASE_URL`: `https://www.bloomberg.com/markets2/api/quote/EQT`

### `ticker_candidates()`

Generate possible Bloomberg ticker candidates for a symbol.

```python
def ticker_candidates(self, symbol: str, mic: str) -> list[str]
```

<ParamField path="symbol" type="str" required>
  Stock symbol (e.g., "D05", "5")
</ParamField>

<ParamField path="mic" type="str" required>
  Market Identifier Code ("XSES" for SGX, "XHKG" for HKEX)
</ParamField>

<ResponseField name="return" type="list[str]">
  List of Bloomberg ticker candidates to try
</ResponseField>

**Example:**

```python
fetcher = BloombergSPCompanyFetcher()

# SGX symbol
sgx_candidates = fetcher.ticker_candidates("D05", "XSES")
print(sgx_candidates)  # ["D05:SP"]

# HKEX symbol (numeric)
hkex_candidates = fetcher.ticker_candidates("5", "XHKG")
print(hkex_candidates)  # ["0005:HK", "5:HK"]

# HKEX symbol (already padded)
hkex_candidates2 = fetcher.ticker_candidates("0700", "XHKG")
print(hkex_candidates2)  # ["0700:HK"]
```

**Notes:**
- For HKEX symbols, generates both zero-padded (4 digits) and non-padded versions
- For SGX symbols, uses `:SP` suffix
- Handles numeric symbols by zero-padding for HKEX

### `fetch_quote_with_candidates()`

Fetch quote data trying multiple ticker candidates.

```python
def fetch_quote_with_candidates(
    self,
    symbol: str,
    candidates: list[str],
    timeout: int = 8,
) -> tuple[Optional[dict], Optional[str]]
```

<ParamField path="symbol" type="str" required>
  Original stock symbol for debug logging
</ParamField>

<ParamField path="candidates" type="list[str]" required>
  List of Bloomberg ticker candidates to try
</ParamField>

<ParamField path="timeout" type="int" default={8}>
  Request timeout in seconds
</ParamField>

<ResponseField name="return" type="tuple[Optional[dict], Optional[str]]">
  Tuple of (quote_data, matched_ticker) where:
  - `quote_data`: Bloomberg quote dictionary or None
  - `matched_ticker`: The ticker that successfully returned data, or None
</ResponseField>

**Example:**

```python
fetcher = BloombergSPCompanyFetcher()

# Generate candidates for HKEX symbol
candidates = fetcher.ticker_candidates("5", "XHKG")

# Try all candidates
quote, matched = fetcher.fetch_quote_with_candidates(
    symbol="5",
    candidates=candidates,
    timeout=10
)

if quote:
    print(f"Matched ticker: {matched}")
    # Process quote data
else:
    print("No matching ticker found")
    # Check debug info
    debug = fetcher.get_last_request_debug()
    if debug:
        print(f"Last error: {debug['error']}")
```

**Notes:**
- Tries each candidate in order until one succeeds
- Stores debug information for each attempt
- Returns on first successful match

### `fetch_background_for_companies()`

Fetch Bloomberg background data for a list of companies.

```python
def fetch_background_for_companies(
    self,
    companies: list[dict],
    mic: str = "XSES",
    delay_per_request: float = 0.1,
    max_workers: int = 1,
    request_timeout: int = 8,
    progress_callback=None,
) -> list[dict]
```

<ParamField path="companies" type="list[dict]" required>
  List of company dictionaries with at least `symbol` and `name` fields
</ParamField>

<ParamField path="mic" type="str" default="XSES">
  Market Identifier Code ("XSES" for SGX, "XHKG" for HKEX)
</ParamField>

<ParamField path="delay_per_request" type="float" default={0.1}>
  Delay in seconds between requests to avoid rate limiting
</ParamField>

<ParamField path="max_workers" type="int" default={1}>
  Number of parallel workers (currently only supports 1 for sequential processing)
</ParamField>

<ParamField path="request_timeout" type="int" default={8}>
  Request timeout in seconds for each API call
</ParamField>

<ParamField path="progress_callback" type="callable" optional>
  Optional callback function `(completed: int, total: int)` for progress updates
</ParamField>

<ResponseField name="return" type="list[dict]">
  List of dictionaries with Bloomberg background data for each company
</ResponseField>

**Example:**

```python
from sgx_company_list import SGXCompanyFetcher
from bloomberg_sp_company_list import BloombergSPCompanyFetcher

# Fetch SGX companies
sgx_fetcher = SGXCompanyFetcher()
companies = sgx_fetcher.get_company_list()

# Fetch Bloomberg background data
bloomberg_fetcher = BloombergSPCompanyFetcher()

def progress(completed, total):
    pct = (completed / total) * 100
    print(f"Progress: {completed}/{total} ({pct:.1f}%)")

results = bloomberg_fetcher.fetch_background_for_companies(
    companies,
    mic="XSES",
    delay_per_request=0.1,
    progress_callback=progress
)

# Process results
for company in results:
    if company['fetch_ok']:
        print(f"{company['symbol']}: {company['name']}")
        print(f"  Market Cap: {company['market_cap']}")
        print(f"  Shares: {company['shares_outstanding']}")
        print(f"  Founded: {company['founded_year']}")
```

**Result Dictionary Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `symbol` | str | Original stock symbol |
| `name` | str | Company name |
| `mic` | str | Market Identifier Code |
| `bloomberg_ticker` | str \| None | Resolved Bloomberg ticker |
| `source_url` | str \| None | Bloomberg API URL used |
| `market_cap` | float \| None | Market capitalization (parsed) |
| `shares_outstanding` | float \| None | Shares outstanding (parsed) |
| `founded_year` | int \| None | Year founded (parsed) |
| `currency` | str \| None | Currency code |
| `fetch_ok` | bool | Whether fetch was successful |

## Usage Example

Complete example demonstrating the full workflow:

```python
from sgx_company_list import SGXCompanyFetcher
from bloomberg_sp_company_list import BloombergSPCompanyFetcher
import json

# Step 1: Fetch SGX companies
print("Fetching SGX companies...")
sgx_fetcher = SGXCompanyFetcher()
sgx_companies = sgx_fetcher.get_company_list(include_null_companies=False)
sgx_companies = [c for c in sgx_companies if c.get("symbol")]
print(f"Found {len(sgx_companies)} SGX symbols")

# Step 2: Fetch Bloomberg background data
print("\nFetching Bloomberg background data...")
bloomberg_fetcher = BloombergSPCompanyFetcher()

def progress(current, total):
    if current % 25 == 0 or current == total:
        pct = (current / total) * 100
        print(f"  Progress: {current}/{total} ({pct:.1f}%)")

results = bloomberg_fetcher.fetch_background_for_companies(
    sgx_companies,
    mic="XSES",
    delay_per_request=0.1,
    request_timeout=8,
    progress_callback=progress
)

# Step 3: Analyze results
with_market_cap = sum(1 for c in results if c.get("market_cap") is not None)
with_shares = sum(1 for c in results if c.get("shares_outstanding") is not None)
with_founded = sum(1 for c in results if c.get("founded_year") is not None)
fetch_success = sum(1 for c in results if c.get("fetch_ok"))

print(f"\nResults Summary:")
print(f"  Total companies: {len(results)}")
print(f"  Successful fetches: {fetch_success}")
print(f"  Market cap coverage: {with_market_cap}/{len(results)} ({with_market_cap/len(results)*100:.1f}%)")
print(f"  Shares coverage: {with_shares}/{len(results)} ({with_shares/len(results)*100:.1f}%)")
print(f"  Founded year coverage: {with_founded}/{len(results)} ({with_founded/len(results)*100:.1f}%)")

# Step 4: Export to JSON
output_file = "bloomberg_companies.json"
with open(output_file, "w", encoding="utf-8") as f:
    json.dump(results, f, indent=2, ensure_ascii=False)

print(f"\nData exported to: {output_file}")

# Step 5: Show sample data
print("\nSample companies with Bloomberg data:")
for company in results[:5]:
    if company['fetch_ok']:
        print(f"\n{company['symbol']}: {company['name']}")
        print(f"  Bloomberg Ticker: {company['bloomberg_ticker']}")
        print(f"  Market Cap: {company['market_cap']:,.0f}" if company['market_cap'] else "  Market Cap: N/A")
        print(f"  Shares Outstanding: {company['shares_outstanding']:,.0f}" if company['shares_outstanding'] else "  Shares: N/A")
        print(f"  Founded: {company['founded_year']}" if company['founded_year'] else "  Founded: N/A")
        print(f"  Currency: {company['currency']}" if company['currency'] else "  Currency: N/A")
```

## Error Handling

The class provides comprehensive error handling:

```python
fetcher = BloombergSPCompanyFetcher()

# Single quote fetch
quote = fetcher.fetch_quote("INVALID")
if quote is None:
    print("Fetch failed")
    debug = fetcher.get_last_request_debug()
    if debug:
        print(f"Error: {debug['error']}")
        print(f"Status: {debug['status_code']}")

# Batch fetch with error tracking
results = fetcher.fetch_background_for_companies(companies)
failed = [c for c in results if not c['fetch_ok']]
print(f"Failed fetches: {len(failed)}")

for fail in failed[:5]:
    print(f"  {fail['symbol']}: No Bloomberg data available")
```

## Rate Limiting

Adjust `delay_per_request` to control request rate:

```python
# Conservative (recommended for large datasets)
results = fetcher.fetch_background_for_companies(
    companies,
    delay_per_request=0.2  # 200ms between requests
)

# Faster (may trigger rate limits)
results = fetcher.fetch_background_for_companies(
    companies,
    delay_per_request=0.05  # 50ms between requests
)
```
